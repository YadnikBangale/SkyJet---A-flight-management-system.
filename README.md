# 🌌 Overview

SkyJet is a modern full-stack **Flight Management System** built using **React.js**, **Spring Boot**, and **MySQL**.

The project provides a premium airline-style dashboard where users can:

✅ Add Flights  
✅ View All Flights  
✅ Search Flights  
✅ Delete Flights  
✅ Manage Flight Routes  
✅ Filter Flights by Price  
✅ Search by Carrier  

The application combines powerful backend engineering with a futuristic glassmorphism UI to create a professional airline management platform.
